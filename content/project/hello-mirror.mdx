<SubTitle>헬로미러란?</SubTitle>
<br/>

헬로미러는 디스플레이에 단순히 정보를 띄우는 것에서 더 나아가 라즈베리파이를 이용하여, 음성 인식 기술로 제어가 가능하며 음성 합성 기술로 사용자에게 피드백을 음성으로 출력하고 각종 데이터를 GUI로 제공해줍니다.

<SubTitle>한계</SubTitle>
<br/>

**1 . 얼굴 인식 기능**

얼굴 인식 기능을 통해 개인을 식별하고 데이터 베이스에 정보를 저장하려 하였으나, 라즈베리파이 자체에서 돌리기에는 성능적인 한계가 있었습니다. 그래서 우리는 그 대안으로 라즈베리파이에서 촬영된 사진을 컴퓨터로 보낸 후 처리 되어진 파일을 컴퓨터에서 라즈베리파이로 다시 전송하는 방식을 구상하였으나, 실력의 한계로 구상의 단계에서 마무리 지어졌습니다.
<br/>

**2. 교내 전시**

마지막 주차 계획은 교내에서 헬로미러를 전시하고 피드백을 받는 것이었습니다. 하지만, 기능을 추가할수록 오류가 늘어났고, 오류를 고치기에 급급했습니다. 따라서 프로젝트 진행속도가 계획보다 더디어져서 완성품을 만드는데 집중하느라, 시연을 통한 피드백은 받지 못하였습니다.
<br/>

**3. 블루투스를 통한 사용자 인식**

블루투스를 통해 사용자의 휴대폰 모델명을 인식하여 사용자를 구분하고자 하였으나 위와 같은 이유로, 다른 기능 구현 과정에서 예기치 않은 변수들이 너무 많이 발생하여, 블루투스 기능을 포기하고 구현하고 있던 기능들의 완성도를 높여야 했습니다.
<br/>

**4. 기차 잔여 좌석 확인 기능**

이전에 계획한 기차 잔여 좌석 확인 기능은 코레일 API 기능상에서 제공되지 않아 아쉽게도 활용할 수 없었습니다.
