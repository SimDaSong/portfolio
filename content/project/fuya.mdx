**개발 배경**
<br/>

<p className="font-bold">프로젝트 구성</p>
<br/>

저희는 Ruby on Rails의 scaffold라는 기능을 프로젝트 작성을 더 쉽게 할 수 있었습니다.
<br/>

<Block>

**Market scaffold model로 만든 것**

: 가게이름(name) 위치(address) 영업시간(open) 주차가능여부(parking-boolean) 전화번호(tel) 휴일(closed) 시/도(locate1) 군/구(locate2)

=> parking 빼고 전부 string
<br/>

**MarketReview scaffold**

Market(상위) - MarketReview(하위)

User(상위) - MarketReview(하위)

model로 만든 것 : 제목(title-string) 내용(content-text) 별점(rate-float) 날짜(date-date)
<br/>

**Shop scaffold**

Market(상위) - Shop(하위)

model로 만든 것 : 가게이름(name) 영업시간(open) 전화번호(tel)
<br/>

**ShopReview scaffold**

Shop(상위) - ShopReview(하위) User(상위) - ShopReview(하위)

model로 만든 것 : 제목(title-string) 내용(content-text) 별점(rate-float) 날짜(date-date)
<br/>

**Menu scaffold**

Shop(상위) - Menu(하위)

model로 만든 것 : 메뉴이름(name) 가격(price-integer) 부가설명(content-text)
</Block>
